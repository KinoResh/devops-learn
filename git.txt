# Git базові команди

## git status
git status
└─ показує що змінилось в папці

## git add
git add .
├─ add   - підготувати файли до коміту
└─ .     - означає "всі файли"

git add notes.txt
└─ notes.txt - тільки цей конкретний файл

## git commit
git commit -m "текст"
├─ commit - зберегти знімок
├─ -m     - далі буде повідомлення
└─ "текст" - опис що ти зробив

## git push
git push
└─ відправити коміти на GitHub

git push -u origin main
├─ -u      - запам'ятати куди пушити
├─ origin  - псевдонім GitHub
└─ main    - яку гілку відправити

## git pull
git pull
└─ скачати зміни з GitHub

## Переглянути
git branch
└─ показати всі локальні гілки

git branch -a
├─ -a - all (всі)
└─ показує і локальні і GitHub гілки

## Створити і перемикатись
git switch <назва>
└─ перейти в гілку

git switch -c <назва>
├─ -c - create (створити)
└─ створити і перейти в гілку

## Перейменувати
git branch -m <стара> <нова>
├─ -m - move/rename
└─ перейменувати гілку

## Видалити
git branch -d <назва>
├─ -d - delete
└─ видалити гілку (якщо злита)

git branch -D <назва>
├─ -D - примусово
└─ видалити навіть якщо не злита

git push origin --delete <назва>
├─ --delete - видалити
└─ видалити гілку на GitHub

## Злити
git merge <назва>
└─ злити гілку в поточну
